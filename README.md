## Install

```bash
$ python -m pip install pip -U
$ python -m pip install ogcapi-pydantic

# or from source
$ git clone https://github.com/developmentseed/ogcapi-pydantic.git
$ cd ogcapi-pydantic
$ python -m pip install -e .
```
